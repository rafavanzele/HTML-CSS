Usando Links Externos

Nesse exemplo você envelopa um termo do seu texto e ativa um link externo usando a tag a. Por exemplo:

Você pode acessar o link do canal da Isabala Boscov clicando aqui

Nesse exemplo você vou deixar linkado o meu perfil no facebook só para exercitar mesmo!

Observação importante: se usar apenas a tag a o link vai abrir na própria página, "fechando" a sua. Para o link abrir numa aba nova, você deve usar a tag target="_blank" e rel="external" 😉

Usando Links Internos

Links internos são ligações (links) de outras páginas (pagina 1, 2, 3 etc) que são também do seu próprio site! Por exemplo:

Você está na primeira página do meu site. Você também pode ir visitar a segunda página desse site.

Você também pode visitar a página de fofocas, clicando na terceira página desse site!

Links para Downloads

Você pode criar links para download, desde que o arquivo a ser baixado esteja dentro da pasta principal do seu projeto HTML. Basta envelopar o termo que for virar link, usar a tag a e dentro do href incluir o arquivo a ser baixado
