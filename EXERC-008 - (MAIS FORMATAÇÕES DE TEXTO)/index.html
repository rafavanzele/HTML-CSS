Código Fonte

O comando CODE é usado para padronizar o tamanho (largura) das letras do seu texto

Inclusive existe um comando (PRE) para formatar código fontes no seu texto. Exemplo:

    
    num = int(input("Digite um numero: ))
    if num % 2 == 0
        print(f'O número {num} é par')
    else
        print(f'O número {num} é ímpar')
